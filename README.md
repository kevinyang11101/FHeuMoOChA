# Python学生选课管理系统 python307

## 项目概述

本项目是基于Python语言和Django框架开发的学生选课管理系统，旨在实现学院、专业、班级、课程、学生和教师信息的有效管理，并提供选课和信息查询功能。

## 技术栈

- **开发语言：** Python
- **框架：** Django
- **前端技术：** HTML、CSS、JavaScript、jQuery
- **后端技术：** Python、Django
- **数据库：** MySQL
- **开发工具：** VSCode、Navicat

## 运行环境

- **Python 版本：** 3.0及以上
- **MySQL 版本：** 8.0

## 功能模块

- **用户模块：**
  - 登录
  - 注册
- **信息管理模块：**
  - 学院信息管理
  - 专业信息管理
  - 班级信息管理
  - 课程信息管理
  - 学生信息管理
  - 教师信息管理
- **选课管理模块：**
  - 选课信息管理
- **查询模块：**
  - 信息查询

## 系统角色

- 学生
- 教师
- 管理员

## 目录结构

```
项目目录/
│
├── apps/             # 应用代码目录
├── static/           # 静态资源目录
├── templates/        # 模板目录
├── manage.py         # 管理脚本
└── 项目名称/          # 项目配置目录
    ├── settings.py
    ├── urls.py
    ├── views.py
    └── wsgi.py
```

## 部署步骤

1. 安装Python环境（版本3.0以上）
2. 安装MySQL数据库（版本8.0）
3. 使用Navicat导入数据库结构
4. 使用VSCode打开项目，安装依赖的Python库
5. 运行`python manage.py runserver`启动开发服务器
